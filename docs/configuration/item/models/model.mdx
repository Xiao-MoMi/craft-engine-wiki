Render an item model.

```yaml
items:
  default:topaz_pickaxe:
    model:
      type: minecraft:model # type is optional in this case
      path: minecraft:item/custom/topaz_pickaxe
      generation:
        parent: "minecraft:item/handheld"
        textures:
          "layer0": "minecraft:item/custom/topaz_pickaxe"
```

:::tip
If you do not specify a type, it will default to using `minecraft:model`.
:::

## Tints

In this configuration, we create a model for custom leaves. However, the default leaves in the vanilla game do not have color, so we need to configure a tint to color them.

```yaml
default:palm_leaves:
  model:
    type: "minecraft:model"
    path: "minecraft:item/custom/palm_leaves"
    generation:
      parent: "minecraft:block/custom/palm_leaves"
    tints:
      - type: "minecraft:constant"
        value: -12012264
```

## How to Create a Dyeable Model

First, create a model with a grayscale texture.
Then, in the preview window, click on the face of the model you want to dye. After that, go to the top toolbar, select the `UV` tab, and check the `Tint` option.
‚ö†Ô∏è Note: You must click the face in the **preview window** first, otherwise the option won‚Äôt be available.

Next, configure the item model:

```yaml
items:
  tints:example:
    model:
      type: minecraft:model
      path: tints:item/example
      generation:
        parent: "tints:block/example"
      tints: # Define the tinted faces you selected in BlockBench
        - type: "minecraft:dye" # First tint, corresponds to faces with tintindex=0 (counting starts at 0)
          value: 144,80,62
        - type: "minecraft:grass" # Second tint, corresponds to faces with tintindex=1
          temperature: 0.5
          downfall: 0.5
```
